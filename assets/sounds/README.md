# Sound Files

This directory should contain the following sound files:

- `correct.mp3` - Sound played when user answers correctly
- `wrong.mp3` - Sound played when user answers incorrectly

## Recommendations:
- Keep sounds short (< 1 second)
- Use soft, pleasant tones
- Ensure files are optimized for mobile (small file size)

## Free Sound Resources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Note: The app will work without sound files, they will just fail silently.
