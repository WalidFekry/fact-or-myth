# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-25

### Added
- Initial release
- Daily question feature with countdown timer
- Free questions mode with multiple categories
- Guest mode (no login required)
- Soft registration (name + avatar only)
- Leaderboard system (top 100 users)
- Comments system for daily questions
- Profile screen with stats and streak
- Dark/Light theme toggle
- Full RTL support for Arabic
- Clean architecture with MVVM pattern
- State management with Provider
- Dependency injection with get_it
- Complete backend API (PHP + MySQL)
- Sample data for testing

### Features
- ✅ One daily question per day
- ✅ Unlimited free questions
- ✅ 5 categories (عشوائي، صحة، معلومات عامة، نفسية، دينية)
- ✅ Guest mode support
- ✅ Simple registration
- ✅ Leaderboard with top 3 highlighting
- ✅ Streak tracking
- ✅ Comments on daily questions
- ✅ Profile management
- ✅ Theme switching
- ✅ Smooth animations
- ✅ Error handling
- ✅ Loading states

### Technical
- Flutter 3.0+

- Clean architecture
